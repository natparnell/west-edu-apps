Indefinite integration (or antidifferentiation) reverses the process of differentiation. If F′(x) = f(x), then F(x) is an antiderivative of f(x).

∫ f(x) dx = F(x) + C

The constant C is needed because differentiating any constant gives zero. So x² + 1, x² + 7 and x² − 3 all differentiate to 2x. The family of antiderivatives differs only by a vertical shift.

Power rule: ∫ xn dx = xn+1/(n+1) + C   (n ≠ −1)

Key results: ∫ sin x dx = −cos x + C,   ∫ cos x dx = sin x + C,   ∫ ex dx = ex + C
